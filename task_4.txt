4. Как убрать синюю рамку вокруг картинки-ссылки?

Вроде у картинки-ссылки в браузерах нет синей рамки, в общем-то как и у текста-ссылки. Синий весь текст с подчёркиванием.

Если же нужно просто убрать рамку, то в CSS для элемента пишем: элемент { border: none; }

Если же нужно убрать подсветку активного поля, например input, то в CSS для элемента пишем: элемент { outline: none; }
